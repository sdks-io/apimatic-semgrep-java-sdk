
# Guidance

Remediation guidance generated by Semgrep Assistant

*This model accepts additional fields of type Object.*

## Structure

`Guidance`

## Fields

| Name | Type | Tags | Description | Getter | Setter |
|  --- | --- | --- | --- | --- | --- |
| `Instructions` | `String` | Optional | Step-by-step instructions explaining to a developer how to fix the finding. AI generated content, review carefully | String getInstructions() | setInstructions(String instructions) |
| `Summary` | `String` | Optional | Short title explaining to a developer how to fix the finding. AI generated content, review carefully | String getSummary() | setSummary(String summary) |
| `AdditionalProperties` | `Map<String, Object>` | Optional | - | Object getAdditionalProperty(String key) | additionalProperty(String key, Object value) |

## Example

```java
import dev.semgrep.ApiHelper;
import dev.semgrep.models.Guidance;
import java.io.IOException;

Guidance guidance = new Guidance.Builder()
    .instructions("instructions4")
    .summary("Use a template rendering engine such as EJS instead of string concatenation.")
.additionalProperty("exampleAdditionalProperty", ApiHelper.deserialize("{\"key1\":\"val1\",\"key2\":\"val2\"}"))
    .build();
```
